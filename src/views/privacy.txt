Данный бот собирает следующие персональные данные:

- Имя, фамилия пользователя
- Имя пользователя телеграмм
- Уникальный Id пользователя телеграмм
- Вышеперечисленные данные далее по тексту Политики объединены общим понятием Персональные данные.

Цели обработки персональных данных:

- Цель обработки персональных данных Пользователя — информирование Пользователя посредством отправки электронных писем; предоставление доступа Пользователю к сервисам, информации и/или материалам, содержащимся в telegram-боте.

Правовые основания обработки персональных данных:

- Оператор обрабатывает персональные данные Пользователя только в случае их заполнения и/или отправки Пользователем самостоятельно через специальные формы, размещенные в Telegram-боте https://t.me/cibm19_bot. Заполняя соответствующие формы и/или отправляя свои персональные данные Оператору, Пользователь выражает свое согласие с данной Политикой.
Оператор обрабатывает обезличенные данные о Пользователе.
